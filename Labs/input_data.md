```

m.jordan_fit_13_24_b_d@knute.edu.ua -> mjordan

```

Тобто:
- візьміть частину електронної пошти **перед `@`**,  
- видаліть такі фрагменти, як `_fit_...`, `_b_d` тощо,  
- видаліть усі символи підкреслення `_`,  
- **результатом є конкатенація ініціалу та прізвища.**
